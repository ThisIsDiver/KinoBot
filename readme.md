# Telegram-бот "КиноБот"  
Это бот, который позволяет искать фильмы и сериалы с помощью **Kinopoisk API**.

---

## Установка и настройка 

### 1. Установите Python  
Для гарантированной работы рекомендуется установить Python версии **3.10 и выше**

---
### 2. Установите зависимости  
Все библиотеки указаны в `requestments.txt`. Установите с помощью команды:  
```bash
pip install -r requestments.txt
```

**Основные используемые библиотеки:**  
- `pyTelegramBotAPI` — библиотека для работы с Telegram Bot API  
- `requests` — библиотека для отправка запросов к Kinopoisk API  
- `peewee` — легковесная ORM-система для создания базы данных и работы с ней.

---

### 3. Добавьте свои токены API

Перед запуском замените строки в `config.json`:  
```python
"TG-TOKEN": "TOKEN",
"Kinopoisk TOKEN": "TOKEN"
```
на соответствующие им токены.

---

## Запуск  

После установки и настройки всего нужного можно запускать бота.  

### Для Linux / macOS:  
```bash
python3 main.py
```

или
```bash
python main.py
```

### Для Windows:  
```bash
py main.py
```
или
```
[Путь до вашего Python] main.py
```

После запуска в консоли появится сообщение:  
```
База данных запущена  
Бот запущен
```

---

## Использование  

1. Откройте Telegram и зайдите в чат своего бота.
2. Нажмите **START**.  
3. В меню выберите нужный режим:  
   - 🔍 **Поиск фильма/сериала по названию**  
   - 💥 **10 случайных топовых фильмов/сериалов по жанру**  
   - 🎲 **Случайный фильм по жанру**  
   - 📜 **История ваших запросов** 

В истории хранится до **15 последних фильмов**, которые вы искали.  

В случае неверной команды, бот будет предлагать нажать на кнопку **/start** и выбрать что-то из меню.

---

## База данных
При первом включении появится файл `history.db`. Это база данных, в которую пишутся запросы пользователей. С помощью неё пользователь может посмотреть все свои последние 15 запросов.
